Home: Team Pursuit Race GA Project

This project aims to develop useful strategies using Genetic Algorithms

The interaction domain chosen is from the world of cycling: the team pursuit track race. This race involves close coordination of efforts between a number of riders of different capabilities.

This project aims to:

  • Model the team pursuit race as a game-like model
  • Model each rider using a set of characteristics that describe their physical abilities
  • Represent race strategies as sequences of instrutions to be given to the lead rider at sspecific points (in time) during the race.
  • With the race itself as a fitness function, use a Genetic Algorithm to evolve useful strategies for any given set of riders.
  • Explore the history of the ideas of evolution and how they are applied in computer science (and more broadly).
